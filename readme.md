## Project Starter for Laravel 5.1

### Features
- Sass
- jQuery
- Bootstrap
- Font-Awesome

### Installation
- `git clone https://github.com/thomasfilius/laravel-project-starter.git {{project-name}}`
- `composer install && npm install && bower install`
- Add <b>.env</b> file
- `php artisan key:generate`
